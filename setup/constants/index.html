Constants

As constants são todas definidas no arquivo src/bootstrap.php podendo ser manipuladas conforme o ambiente e necessidades de seu projeto.

define('DEBUG',true);
define('URL_PREFIX','');
define('REQUEST_URI',$_SERVER['REQUEST_URI']);
define('ROOT_PATH',__DIR__);
define('DATABASE_PATH',ROOT_PATH.'/database.json');
define('DATABASE','default');

DEBUG

define('DEBUG',true);

Enable or disable display errors.

Default: true

Possible values: true or false

URL_PREFIX

define('URL_PREFIX','blog/');

É um prefixo usado para ser escapado nos acessos as URL do projeto.

Example HTTP GET request

GET /blog/ HTTP/1.1

In url.php

// in url.php file the math ^home is valid.
// escaped blog/
'/^\/home\/?$/' => ['Blog/Home/index',['GET']],

Default: ''

Possible values: String

REQUEST_URI

define('REQUEST_URI',$_SERVER['REQUEST_URI']);

É a rota recebida, podendo ser obtida pela global $_SERVER['REQUEST_URI'] do php.

Default: $_SERVER['REQUEST_URI']

Possible values: String

ROOT_PATH

define('ROOT_PATH',__DIR__);

Diretório no qual a aplicação está rodando, podendo ser adquirida pela função __DIR__ do php.

Default: __DIR__

Possible values: String

DATABASE_PATH

define('DATABASE_PATH',ROOT_PATH.'/database.json');

Diretório em que está definido o arquivo json para conexão as bases de dados, geralmente definido como database.json na pasta src/.

Default: ROOT_PATH.'/database.json'

Possible values: String[.json]

DATABASE

define('DATABASE','default');

Nome da base de dados default do projeto, definido no arquivo database.json. Example:

{
    "default": {
        "driver": "sqlite",
        "host": "/home/user/project/database/db_restaurant",
        "name": "",
        "user": "",
        "password": "",
        "port": "",
        "autocommit": 0,
        "debug": 0
    }
}

Possible values: String

Default: default
