Configuration

For running, open the console and runner file with command ./runserver.sh in the directory where the project. Willer not need a web server to run, taking advantage of the built-in php service.

Requisite

  • You must use the PHP version >= 5.6
  • Linux(obvious)

PHP server built-in

Execute command ./runserver.sh

Nginx

Enable URL rewrite(is required), on file default and also in other in your sites-avaliable path.

location /willer {
    rewrite ^/willer/(.*) /willer/bootstrap.php last;
}

replace willer by the path where its application will run. The bootstrap.php file is ha index for willer.

php5-fpm

configure the php for php-fpm.

location ~ \.php$ {
    include snippets/fastcgi-php.conf;
    fastcgi_pass unix:/var/run/php5-fpm.sock;
}
