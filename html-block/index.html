Willer Framework

Html Block

Html block it's the interface creator designed for Willer, created to assemble dynamic components in HTML.

This library mounts interfaces using bootstrap css markup

Components

$html_block = new HtmlBlock\HtmlBlock();

$html_block_render = $html_block
    ->setHeadTitle('System title')
    ->addMeta(['charset' => 'utf-8'])
    ->addCss('bootstrap.min.css')
    ->addJs('jquery-3.1.1.min.js');

$html_block_render->appendBody();
$html_block_render->appendBodyRow([]);
